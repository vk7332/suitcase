# SUITCASE Affiliate Integration

## Overview
This module provides a complete affiliate and referral system for the SUITCASE LegalTech platform.

## Included Components
- ReferralLinkCard
- EarningsCard
- ReferralsTable
- PayoutRequestForm

## Utilities
- referralUtils.ts

## Installation
1. Extract this ZIP into your project root.
2. Install dependencies:
   npm install
3. Run the project:
   npm run dev

## Features
- Referral link generation
- Earnings tracking
- Referral management
- Payout requests
- Modular and scalable design
